Rilis Pers Asli

29 SEPTEMBER 2016: PostgreSQL 9.6, versi terbaru sistem database relasional Open Source tercanggih dunia, dirilis hari ini oleh PostgreSQL Global Development Group. Rilis ini akan memungkinkan pengguna untuk melakukan baik scale up dan scale out beban kerja database yang berperforma tinggi. Fitur-fitur baru mencakup query secara parallel, peningkatan dalam replikasi synchronous, pencarian frase, dan peningkatan pada performa dan kegunaan.

Scale Up dengan Query Parallel

Versi 9.6 menambah dukungan untuk paralelisasi beberapa operasi query, memungkinkan penggunaan sebagian atau seluruh jumlah core dalam server untuk menampilkan hasil query lebih cepat. Rilis ini mencakup scan (table) paralel berurut, agregasi, dan join. Bergantung pada detil dan jumlah cores yang ada, paralelisme dapat mempercepat query-query data besar hingga 32 kali lebih cepat.

"Saya memigrasikan seluruh platform data genomics kami — seluruh 25 milyar baris MySQL legacy — ke satu database Postgres tunggal, mengedepankan kemampuan kompresi baris dari tipe data JSONB, dan mode index GIN, BRIN, dan B-tree yang hebat. Kini dengan versi 9.6, Saya berharap untuk menggunakan fungsionalitas query paralel untuk memungkinkan skalabilitas yang lebih besar lagi untuk query terhadap tabel-tabel kami yang berukuran besar," sebut Mike Sofen, Chief Database Architect, Synthetic Genomics.

Scale Out dengan Replikasi Synchronous dan postgres_fdw

Dua opsi baru telah ditambahkan ke fitur replikasi synchronous di PostgreSQL yang memungkinkannya untuk digunakan dalam menjaga bacaan data yang konsisten di sepanjang kluster database. Pertama, fitur ini kini memungkinkan konfigurasi kelompok-kelompok replika synchronous. Kedua, mode "remote_apply" membuat tampilan data yang lebih konsisten di sepanjang node-node berbeda. Fitur-fitur ini mendukung penggunaan replikasi built-in untuk menjaga sekumpulan node "identik" untuk melakukan load balancing pada beban kerja bacaan.

Driver untuk federasi data PostgreSQL-ke-PostgreSQL, postgres_fdw, punya kemampuan-kemampuan baru untuk melaksanakan tugas server-server secara remote. Dengan mem"push down" sort, join, dan update data secara batch, pengguna dapat mendistribusikan beban kerja di sepanjang beberapa server PostgreSQL. Fitur-fitur ini akan segera ditambahkan ke driver-driver FDW lainnya juga.

"Dengan kemampuan-kemampuan JOIN, UPDATE dan DELETE secara remote, Foreign Data Wrappers kini menjadi solusi lengkap untuk berbagi data antara database lain dengan PostgreSQL. Contohnya, PostgreSQL dapat digunakan untuk menangani input data menuju ke dua atau lebih jenis database berbeda," sebut Julyanto Sutandang, Director of Business Solutions at Equnix.

Pencarian teks Lebih baik Dengan Frase

Fitur pencarian teks penuh di PostgreSQL kini medukung "phrase search." Ini memungkinkan pengguna mencari frase secara persis, atau kata-kata dalam kedekatan yang ditentukan di antaranya, menggunakan index GIN yang cepat. Dikombinasikan dengan fitur-fitur baru untuk fine-tuning opsi pencarian teks, PostgreSQL adalah pilihan superior untuk "hybrid search" yang menggabungkan relasional, JSON, danpencarian teks penuh.

Penggunaan Lebih Halus, Cepat dan Mudah

Dengan feedback dan pengujian oleh pengguna-pengguna PostgreSQL dengan database produksi bervolume tinggi, proyek database telah mampu meningkatkan banyak aspek performa dan kegunaan dalam rilis ini. Replikasi, agregasi, indexing, sorting, dan stored procedure telah dibuat lebih efisien, dan PostgreSQL kini mempergunakan sumber daya kernel Linux dengan lebih baik. Overhead administrasi untuk tabel-tabel besar dan beban kerja complex juga dikurangi, khususnya lewat pengikatan pada VACUUM.

Lisensi

PostgreSQL menggunakan Lisensi PostgreSQL, suatu lisensi BSD-like yang "permisif". Lisensi OSI-certified ini banyak diapresiasi karena bersifat fleksibel dan ramah terhadap bisnis, karena tidak melarang penggunaan PostgreSQL dengan aplikasi komersil dan berbayar. Bersama dukungan banyak perusahaan dan kepemilikan publik dari source code, lisensi kami membuat PostgreSQL sangat populer dengan vendor yang ingin menanamkan database dalam produk mereka tanpa takut biaya, vendor lock-in, atau perubahan dalam syarat-syarat lisensi.

Tentang PostgreSQL

PostgreSQL adalah system database tercanggih dunia, dengan komunitas global ribuan pengguna dan kontributor dan lusinan perusahaan dan organisasi. Proyek PostgreSQL membanguna di atas lebih dari 30 tahun pengembangan, yang dimulai dari University of California, Berkeley, dan memiliki kecepatan pengembangan yang tak tertandingi kini. Kumpulan fitur PostgreSQL hyang matang tidak hanya menyamai sistemm database berbayar populer, namun juga melebih mereka dalam fitur-fitur database tingkat lanjut, ekstensibilitas, keamanan dan stabilitas. Pelajari lebih lanjut tentang PostgreSQL dan bergabunglah dalam komunitas kami di PostgreSQL.org.
